# 用户操作手册

## 1. 系统概述

本系统是一个功能完备的待办事项管理系统，旨在帮助用户有效地管理任务、跟踪进度和提高工作效率。系统支持任务的创建、编辑、删除、状态更新，以及任务依赖关系管理、甘特图可视化、数据统计等功能。

## 3. 系统登录

### 3.1 注册新用户
1. 打开系统登录页面
2. 点击"注册"按钮
3. 填写用户名、邮箱和密码
4. 点击"注册"按钮完成注册

### 3.2 用户登录
1. 打开系统登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后，系统会自动跳转到任务列表页面

### 3.3 用户登出
1. 点击页面右上角的用户头像
2. 选择"登出"选项
3. 系统会自动跳转到登录页面

## 4. 任务管理操作

### 4.1 创建任务
1. 登录系统后，进入任务列表页面
2. 点击"新建任务"按钮
3. 填写任务标题、描述、优先级、状态、开始时间、结束时间等信息
4. 选择任务负责人（可选）
5. 点击"保存"按钮

### 4.2 编辑任务
1. 在任务列表页面，找到要编辑的任务
2. 点击任务右侧的"编辑"按钮
3. 修改任务信息
4. 点击"保存"按钮

### 4.3 删除任务
1. 在任务列表页面，找到要删除的任务
2. 点击任务右侧的"删除"按钮
3. 在弹出的确认对话框中点击"确认"按钮

### 4.4 更新任务状态
1. 在任务列表页面，找到要更新状态的任务
2. 点击任务状态下拉菜单
3. 选择新的状态
4. 系统会自动保存状态变更

## 5. 任务依赖关系管理

### 5.1 创建任务依赖关系
1. 进入任务详情页面
2. 找到"依赖关系"部分
3. 点击"添加依赖"按钮
4. 选择依赖的任务
5. 点击"保存"按钮

### 5.2 删除任务依赖关系
1. 进入任务详情页面
2. 找到"依赖关系"部分
3. 点击依赖关系右侧的"删除"按钮
4. 系统会自动删除依赖关系

## 6. 数据可视化

### 6.1 甘特图视图
1. 点击左侧导航栏中的"甘特图"选项
2. 系统会显示所有任务的甘特图视图
3. 可以通过拖拽调整任务的时间范围
4. 可以查看任务之间的依赖关系

### 6.2 任务时间线
1. 点击左侧导航栏中的"时间线"选项
2. 系统会显示任务的时间线视图
3. 可以通过日期选择器设置时间范围
4. 可以查看指定时间范围内的任务

### 6.3 数据统计
1. 点击左侧导航栏中的"统计"选项
2. 系统会显示任务完成情况统计
3. 可以查看任务状态分布、用户任务分布等信息
4. 可以查看已过期和即将过期的任务

## 7. 管理员操作

### 7.1 查看所有用户
1. 登录系统（需要管理员权限）
2. 点击左侧导航栏中的"用户管理"选项
3. 系统会显示所有用户的列表
4. 可以查看每个用户的详细信息

## 8. 常见问题

### 8.1 忘记密码
目前系统暂不支持密码重置功能，请联系管理员重置密码。

### 8.2 任务无法删除
请检查是否有其他任务依赖于该任务，如果有，请先删除依赖关系。

### 8.3 甘特图不显示任务
请检查任务是否设置了开始时间和结束时间，甘特图只显示有时间范围的任务。

### 8.4 无法创建依赖关系
请检查依赖的任务是否存在，以及是否已经存在相同的依赖关系。
